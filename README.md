# 💍 Wedding Management System

## 📌 Overview  
Developed a console-based Wedding Management System in Java to simplify event planning and booking.  
The system allows users to manage wedding events efficiently while providing an admin panel for monitoring, control, and analytics.

## 🚀 Features  
- User Authentication (Login, Signup, Forgot Password)  
- Multi-event booking (Sangeet, Mehendi, Reception, Wedding)  
- Real-time venue availability management  
- Service selection (Catering, Decoration, Photography)  
- Add-ons and coupon-based discounts  
- Booking cancellation with waitlist system  
- Invoice generation with GST calculation  

## 🛠️ Admin Panel  
- View and manage all bookings  
- Update booking status  
- Manage venues and coupons  
- Access customer records  
- View analytics (revenue, popular venue, peak month)  
- Export financial reports  

## 🧠 Tech Stack  
- Java (Core Java, OOP)  
- Data Structures (ArrayList, HashMap)  
- File Handling  
- Java Stream API  
 
## ▶️ Run the Project  
javac WeddingManagement.java  
java WeddingManagement  

## 💡 Key Learnings  
- Applied object-oriented design in a real-world project  
- Managed dynamic data using collections  
- Built role-based system (User & Admin)  
- Implemented validation and structured workflows
